Instruction

1. Once a rating window pops up, select the preferred video from the video pair. Select the "left" button if the left video is preferred, select the "right" button if the right video is preferred, and select "not sure" if you have no preference between two videos. When you have selected the preferred video, click the "Next" button.

2. If you do not select the preferred video within 5 seconds, a window pops up, asking you to make your choice within 3 seconds. You must select one of the options ("left", "right", or "not sure"). If you do not make your choice, the question will remain unanswered, and you will be redirected to the next question.

3. There are some test questions in each hit to control the quality of the subjective test. Please answer all questions carefully.

4. If you fail to answer more than two questions or answer a test question incorrectly, the entire hit will be rejected.

5. After the subjective test is completed in the current hit, submit the result. Then you can start with the next hit.
